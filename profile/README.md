# .github
欢迎来到 **MATS Studio**（隶属于 BriSpike 联合体），这是一个以数理科学、算法研究、程序设计为核心发展方向，现阶段以游戏编程开发为实践载体的跨领域技术创作组织！本组织是我们的官方组织.

我们现阶段聚焦游戏编程开发与技术实践，同步围绕 Minecraft、Terraria、Undertale 、I Wanna等作品开展技术向同人创作与交流；未来将逐步深化一般数学研究、算法创新与多领域程序设计实践，持续支持数学 / 物理小程序、操作系统模拟器等各类技术作品的研发、分享与落地，以技术实践夯实核心学科能力，以学科研究反哺技术创作.

在本组织内，你可以参与算法研讨、程序设计交流、游戏开发技术实践，也可以发布数理研究心得、算法优化成果、各类程序开发作品及技术向同人创作内容.我们非常鼓励大家探索创新、突破思维边界，让知识从理论走向实践！


**目前本组织的可入选作品类型：**

①MATS Studio作品，包括协作作品和成员个人作；

②对开放改编/二创的MATS Studio作品的改编/二创作品.

**有下列情况则不予采纳：**

①作品质量过低，或改编/二创作品缺乏创新；

②包含敏感、违规或恶意引战的内容；

③含有其它工作室标识或其它广告宣传类内容.

我们的发展体系为 “一核二基六翼”，贴合核心发展方向与阶段实践需求：以数理科学为核心，算法与程序设计为基础，覆盖算法研究、程序开发、游戏编程、数据分析、信息科学、自然科学六大实践方向.
